本ページは、労務・人事制度 要件DB（v0_1）において提供されるイベント別CSVの位置づけおよび利用上の前提を説明するための共通READMEです。

配布物について

まず、本ページでは、労務・人事制度 要件DB（v0_1）に含まれるイベント別CSVを配布しています。

次に、イベント別CSVは、特定の業務イベントごとに、対応する要件行のみを抽出して収録したCSVです。

版について

まず、本ページで配布するイベント別CSVの版は v0_1 です。

ファイル名規約

まず、イベント別CSVのファイル名は、以下の形式としています。

  • labor-hr_audit_checklist_<event_suffix>_v0_1.csv

次に、<event_suffix> は、イベントを識別するための末尾識別子です。

例として、HIRE01 などを含みます。

event_id と event_name について

まず、各イベント別CSVに含まれる event_id および event_name は、正本となる包括CSVに記載された値に基づいています。

次に、イベント別CSVの一覧ページでは、event_id の昇順で表示しています。

そのため、表示順は、重要度や優先度を示すものではありません。

列構成および check_status 列について

まず、イベント別CSVの列構成は、包括CSVと完全に同一です。

次に、両者の違いは、行の整理単位のみです。

そのため、各列の役割や check_status 列の利用方法については、包括CSVのREADME（v0_1）に記載された説明を参照してください。

利用上の前提について

まず、イベント別CSVの利用上の前提、想定される利用シーン、提供しないこと、利用条件、免責事項については、包括CSVのREADME（v0_1）にまとめて記載しています。

そのため、これらの詳細は、包括CSVのREADMEを参照してください。
