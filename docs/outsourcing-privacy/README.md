# 委託先管理・個人情報保護 要件DB（v0.1）

本リポジトリは、委託先管理および個人情報保護に関する制度上の要件を、
**判断・解釈・推奨を一切行わず**、
一次情報に基づいて構造化し、参照可能な形で提供するためのデータベースである。

---

## 本DBの設計方針

本DBは、次の原則に基づいて設計されている。

- 判断・推奨・最適解・評価・優劣比較を一切行わない。
- 記録対象は、制度上の要件・義務・位置づけ・参照元に限定する。
- 一次情報（法令、政省令、告示、公式ガイドライン、官公庁・準公的機関の公開資料）のみを使用する。
- 解説記事、ブログ、Q&A、実務解釈は収録しない。
- 更新は best effort とし、網羅性や即時反映を保証しない。

本DBは、委託可否や適法性を判断するためのものではなく、  
**判断前に「何が要求されているか」を確認するための参照DB**である。

---

## 初期収録イベント（v0.1）

v0.1 では、次の3つのイベントを収録している。

1. **SaaS利用開始**  
   クラウドサービスを業務で利用開始する場面。

2. **外注先への個人情報提供**  
   外部委託先に個人情報を提供する場面。

3. **再委託**  
   委託先がさらに第三者へ業務を再委託する場面。

各イベントについて、
「確認が求められている要件」と「その根拠」を、
1行＝1要件として記録している。

---

## データ構成について（重要）

本リポジトリの `data/` ディレクトリに含まれるCSVファイルは、

- 構造例の提示
- 設計検証
- 参照用サンプル

を目的とした **公開サンプルデータ**である。

### 注意事項

- 本CSVは、完全性・網羅性・最新性を保証するものではない。
- 本CSVは、正本データや有料提供データを示すものではない。
- 更新保証や横断的な完全データ提供は、本リポジトリのスコープ外である。

---

## 利用上の注意

- 本DBは法的助言や判断を提供するものではない。
- 実務上の判断は、必ず事業者または専門家の責任において行うこと。
- 本DBの内容を利用した結果について、作成者は責任を負わない。
